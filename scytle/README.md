# Scytle

**Live at:** [https://scytle.com](https://scytle.com)

Scytle helps you go from AI draft to shippable page in one editable workspace.

## Overview
Generate in Chat, organize in Files, polish in Design, then share or export to ship. Scytle is a complete design system in one editable workspace.

- **Draft:** Use Vertex AI Gemini models to structurally generate high-quality wireframes.
- **Refine:** A powerful integrated Canvas with exact component controls, auto-save, and complete ReactFlow state.
- **Ship:** Polish and distribute your designs immediately or export to code.

## Tech Stack
- **Framework:** Next.js 16 (React 19 compiled)
- **Styling:** Tailwind CSS 4 & Radix / shadcn UI
- **State Management:** Zustand 5 & Immer
- **Database & Auth:** Appwrite
- **AI:** Google Vertex AI (Gemini 2.5 Pro / Flash)

## Getting Started

``bash
cd scytle
npm install
npm run dev
``
